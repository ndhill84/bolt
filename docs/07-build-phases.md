# Build Phases

## Phase 1 — Foundation (Day 1-2)
- Scaffold frontend + backend + shared types.
- Setup Postgres + Prisma schema + initial migrations.
- Story CRUD and board list endpoints.

## Phase 2 — Core Workflow (Day 3-4)
- Board UI with drag/drop.
- Story drawer with notes and dependencies.
- Blocked status logic.

## Phase 3 — Context + Activity (Day 5-6)
- File uploads via presigned URLs.
- Agent session/event/artifact APIs.
- Agent Activity UI with live-ish updates.

## Phase 4 — Digest + Hardening (Day 7)
- Sprint/project digest endpoints.
- Validation, pagination, error handling.
- Basic tests and deployment docs.
